# Live QR Code Generator

A real-time QR code generator built with React, TypeScript, and Material-UI. Generate QR codes instantly as you type!

## 🚀 Features

- **Real-time generation** - QR codes appear as you type
- **Instant download** - Save QR codes as PNG files
- **Responsive design** - Works on desktop and mobile
- **Clean interface** - Modern Material-UI design
- **Fast performance** - Optimized with Vite

## 🎯 Live Demo

Visit the live application: [https://codejedi-ai.github.io/qr-code-generator/](https://codejedi-ai.github.io/qr-code-generator/)

## 🛠 Technology Stack

- **React 18** - Modern React with hooks
- **TypeScript** - Type-safe development
- **Material-UI v7** - Beautiful UI components
- **Vite** - Fast build tool and dev server
- **QRCode.js** - QR code generation library

## 🏃‍♂️ Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/codejedi-ai/qr-code-generator.git
   cd qr-code-generator
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173`

## 📱 How to Use

1. **Start typing** any URL in the input field
2. **Watch the QR code generate** in real-time (300ms debounce)
3. **Download the QR code** as a PNG file
4. **Clear and start over** anytime

## 🔧 Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## 📦 Build and Deploy

This project is automatically deployed to GitHub Pages using GitHub Actions. Every push to the `main` branch triggers a new deployment.

To build locally:
```bash
npm run build
```

The built files will be in the `dist/` directory.
