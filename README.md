# Happily Ever After City

Happily Ever After City is a futuristic nightlife-themed concept website built with React, TypeScript, and Material UI.

## Overview

This site presents a high-energy, neon-styled proposal experience with:

- A cinematic hero section
- City imagery cards and nightlife concept blocks
- Step-by-step relationship journey flow
- Clear consent, legal compliance, and anti-trafficking safeguards

## Tech Stack

- React 18
- TypeScript
- Material UI v7
- Vite

## Run Locally

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Scripts

- npm run dev
- npm run build
- npm run preview
